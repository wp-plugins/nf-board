== Description ==

NF BOARD [엔플린트 보드](http://nfboard.co.kr) 는 워드프레스 기반의 플러그인 형태로 작동하는 
누구나 친숙하게 편리하게 이용이 가능한 한국형 회원관리 & 게시판 통합보드 플러그인 입니다.

* 워드프레스 한국형 게시판 회원관리&게시판 통합 관리 솔루션

= NF board 회원관리 & 게시판 특장점 =

* 회원관리 기능
워드프레스 회원관리와 100% 연동 
회원가입시 입력항목 설정 기능
이용약관, 개인정보 취급방침 설정
회원가입후 메일 보내기 설정 기능 제공 
기본 도로명 주소 등록및 공공데이터 포털, Daum 우편번호 API 연동 가능

* 게시판 기능
최신글 제공
자동글 등록방지 기능
게시물 필터링 기능 
게시판 권한설정 기능 (비회원, 회원, 관리자)
관리자 다중 셀렉트 삭제 
댓글 기능,비밀글 기능, 공지기능, 파일첨부 기능
비회원 개인정보 수집및 활용동의 기능
게시글 작성시 워드프레스 에디터 설정가능

Shortcode(숏 코드) 제공 
로그인, 회원가입/정보수정, 아이디찾기, 비밀번호 찾기, 최신글, 게시판 을
페이지내에 원하는 부분에 단축 코드를 붙혀 넣을수 있기 때문에 쉽게 페이지 디자인 가능

사용자가 자유롭게 원하는 스타일로 작업가능한 스킨 형태의 파일로 선택제공

== Installation ==


1. 사용중인 계정내에 FTP 서버에 접속
2. 워드프레스 설치 가 설치된 플러그인 디렉토리 `/wp-content/plugins/` 에 파일 업로드 후 압축 해제 
3. 워드프레스 관리자 로그인 이후에 관리 페이지에서 플러그인 -> 설치된 플러그인 -> NF BOARD 플러그인 활성화


== Frequently Asked Questions ==

= Q : NF BOARD 작동 환경서버 를 알고 싶습니다. =


NF BOARD 는 워드프레스 가 설치되어 있는 환경이라면 100% 지원을 합니다. 현재 워드프레스 최신버젼인 4.x 에서 지원을 합니다.

= Q : NF BOARD 스킨을 수정하고 싶습니다. 스킨파일 위치 가 어디인가요 ? =


/워드프레스 설치된 경로/wp-content/plugins/NF-Board/templates/board/ 게시판 스킨
/워드프레스 설치된 경로/wp-content/plugins/NF-Board/templates/Member/ 회원관리 스킨

위에 경로에 접속을 한다음 Default 기본 스킨 파일의 이름을 변경 하고 사용자 임의적으로 CSS 스타일을 수정하거나 php 파일을 수정 하면 됩니다.

= Q : NF BOARD 설치 방법을 알고 싶습니다 ? =


FTP 서버에 접속을 하여 /워드프레스 설치된 경로/wp-content/plugins 디렉토리 내에
NF-barod_*.zip  파일을 업로드 한다음 플러그인 페이지내에서 NF BOARD “활성화”
을 해주시면 됩니다.
